# v1.0.4 发布说明

## 本次发布

- Windows 桌面端：PyQt5 检测界面，支持默认检测、精细扫描、手动框选和本地结果归档。
- Android 客户端：版本 1.0.4，最低 Android 14/API 34，目标 SDK 36，包含离线检测模型和结果同步队列。
- 服务器与模型工具源码同步进入仓库，便于后续发布和模型一致性校验。

## 安装和安全提示

- Windows 用户应优先下载完整 ZIP；ZIP 内的 EXE 必须与 `_internal` 目录、模型文件和 `release.json` 一起使用。
- APK 已通过 APK Signature Scheme v2 校验，但当前使用 Android Debug 证书，不是正式商店签名。正式分发前应使用受保护的发布密钥重新签名，并把密钥和密码留在仓库之外。
- Android 设备需要 Android 14/API 34 或更高版本。
- 下载完成后请使用同一 Release 中的 `SHA256SUMS.txt` 校验文件完整性。
