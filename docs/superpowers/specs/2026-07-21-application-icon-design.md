# 水体反应管检测应用图标设计规格

状态：已由用户确认方向与视觉规格，待实现验收。

## 1. 背景与目标

为“水体反应管检测”Android 客户端、Windows 安装程序和官网直下发布页提供统一应用图标。图标需要在手机桌面、Windows 安装包、网站下载入口等不同尺寸下保持清晰，并延续项目已有的档案化实验室视觉系统。

## 2. 已确认设计方向

采用“反应管信号（Reaction Signal）”方向：

- 青色背景承载应用识别色。
- 中央为白色反应管，底部使用荧光黄表示反应液。
- 黑色刻线和中心定位点表达检测、判读与校准。
- 左右使用简化扫描波纹，表达图像识别和检测信号。
- 不使用文字、字母、数字、渐变、复杂阴影或装饰性背景。
- 采用扁平、几何、高对比、强轮廓的视觉语言。

## 3. 视觉令牌

| 令牌 | 值 | 用途 |
| --- | --- | --- |
| Signal Cyan | `#00A7B5` | 主背景、品牌识别 |
| Signal Yellow | `#D9FF3F` | 反应液、强调信号 |
| Ink | `#111111` | 刻线、定位点、轮廓 |
| Paper | `#FFFFFF` | 反应管主体 |

## 4. 资源交付

正式实现应生成以下资源：

- Android 自适应图标前景资源和背景资源，遵守 Android 安全区域，避免反应管或扫描波纹被圆形、圆角或异形蒙版裁切。
- Windows 多尺寸 `.ico`，至少覆盖安装程序和桌面快捷方式使用的常见尺寸。
- 统一的正方形 PNG 预览，用于官网下载区、GitHub Release 说明和中文使用说明。
- 必要时提供 SVG 源稿或可追溯的矢量源，以便后续修改颜色和尺寸。

## 5. 接入范围

1. Android：替换 Launcher 图标资源，保持现有应用包名和启动逻辑不变。
2. Windows：将 ICO 接入 Inno Setup 安装程序及快捷方式配置，不改变桌面检测程序的运行行为。
3. 官网直下：将 PNG/ICO 预览用于下载入口和发布说明，不在图标中嵌入版本号，避免每次发布重新绘制。
4. GitHub：只提交公开图标资源和文档，不提交 Android 签名密钥、Windows 证书、密码或服务器令牌。

## 6. 验收标准

- 48dp 左右的 Android 桌面图标仍能看出“反应管 + 反应液”的主体关系。
- Windows 小图标不依赖文字也能与项目其他发布素材保持一致。
- 在浅色、深色和系统异形蒙版下，主体不被裁切，颜色对比清楚。
- Android、Windows、官网和发布说明使用同一版图标，不出现不同颜色或不同轮廓的分叉版本。
- 资源文件不包含水印、额外文字或第三方商标。

## 7. 不在本次范围内

- 不修改应用名称、包名、服务器协议或检测模型。
- 不制作独立品牌字标或完整企业 VI 手册。
- 不将用户照片、检测结果或服务器数据嵌入图标素材。
