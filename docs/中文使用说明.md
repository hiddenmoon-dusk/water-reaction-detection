# 水体反应管检测系统中文使用说明

## 一、Windows 电脑端

### 安装

1. 在 GitHub Releases 下载 `水体反应管检测系统-v1.0.4-win64.zip`。
2. 将 ZIP 解压到有写入权限的本地目录。
3. 保持以下文件在同一个程序目录中：
   - `水体反应管检测系统.exe`
   - `reaction_classifier.h5`
   - `yolov8n.pt`
   - `release.json`
4. 双击 EXE 启动程序。首次启动可能需要等待模型加载。

### 检测流程

1. 选择水样类型：污水、生活用水或养殖水体。
2. 点击图片选择区域，选择一张或多张水样照片。
3. 选择检测方式：
   - **默认检测**：适合反应管大小正常、背景较简单的图片。
   - **精细扫描**：适合大尺寸照片或反应管较小、默认检测可能漏检的图片。
   - **手动框选**：拖拽框出需要分析的反应管，适合目标遮挡或背景复杂的图片。
4. 等待结果显示，确认“已反应/未反应”和置信度。
5. 点击保存结果。程序会按水样类型和检测批次建立本地归档。

### 网络同步

电脑端会把结果先保存到本地。服务器暂时不可达时，任务会进入待上传队列并自动重试；不要为了“清空提示”而删除队列文件。若服务器发布代次发生变化，旧客户端可能需要重新安装最新版本。

## 二、Android 手机端

### 安装条件

- Android 14/API 34 或更高版本。
- 允许应用使用相机和照片/媒体文件。
- APK 已内置检测模型，普通检测不要求实时联网。

### 检测流程

1. 安装 `水体反应管检测-v1.0.4.apk`。
2. 启动后选择水样类型。
3. 选择“拍照”或“从相册选择”。
4. 进入检测页并等待检测完成；精细扫描会显示进度。
5. 查看结果，确认后保存到本地记录。
6. 网络恢复后，应用会自动上传待同步结果；上传成功的记录才会从待上传队列中清理。

### 重要提示

- 不要安装来源不明或被重新打包的 APK。
- 若系统提示版本过低，请确认设备是否为 Android 14/API 34 以上。
- 若检测结果页面提示模型或发布版本过期，请从 Releases 下载最新 APK。
- 拍摄时尽量让反应管完整、光线均匀，避免强反光和严重遮挡。

## 三、结果与隐私

- 原始照片、标注图片和结果 JSON 可能包含现场信息，请按单位数据管理要求保存和传输。
- 本地结果目录、服务器数据库、上传归档和测试样本不应提交到 GitHub。
- 服务器管理员密码只应通过服务器环境变量或受保护的密钥文件提供，不能写进代码、截图或说明文档。

## 四、校验下载文件

在 Windows PowerShell 中进入 Release 资产所在目录后执行：

```powershell
Get-FileHash .\水体反应管检测-v1.0.4.apk -Algorithm SHA256
Get-FileHash .\水体反应管检测系统-v1.0.4.exe -Algorithm SHA256
Get-FileHash .\水体反应管检测系统-v1.0.4-win64.zip -Algorithm SHA256
```

将输出结果与同一 Release 中的 `SHA256SUMS.txt` 比对，哈希一致后再安装或运行。
