# 水体反应管检测系统

版本：v1.0.5
发布主体：个人
官网：<https://hiddenmoon.duckdns.org/>

## 一、功能简介

本系统根据水样反应管照片进行辅助识别，并提供结果保存和可选的服务器同步。检测结果仅供技术辅助使用，不能替代实验室检测、人工复核或监管结论。

## 二、Windows 电脑端

1. 下载 `water-detection-desktop-v1.0.5-win64.zip`，解压到本地目录。
2. 双击目录中的 `水体反应管检测系统.exe` 启动程序。不要只复制 exe，需保留同目录的 `_internal`、模型文件和 `release.json`。
3. 选择水样类型，再选择照片或拍照文件；程序会先进行默认检测。
4. 在检测结果页检查识别框和标签；需要时使用精细扫描或手动框选。
5. 保存结果后，可在联网时点击上传。上传前请确认照片中没有身份证件、面部、住址、联系方式或其他个人隐私。

Windows 便携包暂未使用商业代码签名证书，首次运行时 Windows SmartScreen 可能显示“未知发布者”。请从官网或项目发布页下载，并用同页提供的 SHA-256 校验值核对文件。

## 三、Android 手机端

1. 下载 `water-reaction-android-v1.0.5.apk`，在 Android 14 或更高版本设备上安装。
2. 按系统提示授予相机和照片选择所需的权限；不使用相关功能时可以拒绝权限。
3. 选择水样类型，拍照或从相册选择照片，确认后等待检测完成。
4. 检查结果后选择保存；联网时可以使用上传功能同步结果。
5. APK 只应从官网或项目发布页下载。安装更新时请确认包名为 `org.hiddenmoon.waterreaction`，不要安装来源不明的同名应用。

## 四、数据与隐私

检测结果和图片会一同上传到服务器。服务器数据通常仅在服务器保存一天，主要用于优化识别模型以及相关检测结果的统计。请务必在上传前自行确认照片中不含个人隐私。

如需申请删除与你有关的数据，请发送邮件至 [sunx77@mail2.sysu.edu.cn](mailto:sunx77@mail2.sysu.edu.cn)，并提供上传时间、结果标识或其他可定位信息。完整政策见官网的[用户隐私政策](https://hiddenmoon.duckdns.org/privacy)，使用前请阅读[用户协议](https://hiddenmoon.duckdns.org/terms)。

## 五、常见问题

### 程序无法启动

请确认 Windows 解压完整，模型文件和 `_internal` 目录没有被移动或杀毒软件隔离；不要从压缩包内直接运行。

### 检测结果不准确

请重新拍摄清晰、光线均匀、反应管完整可见的照片，并在重要场景中进行人工复核或实验室复检。

### 上传失败

请先确认网络可访问官网，再检查应用版本是否为当前发布版本。离线检测结果可以先保存在本地，稍后重新上传。

### 删除数据或反馈问题

请联系 `sunx77@mail2.sysu.edu.cn`。反馈时只需提供时间、版本和错误现象，不要在邮件中发送密码、私钥或服务器 Token。
